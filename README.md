# Counter-Strike 透视自瞄教程

一款针对`Counter-Strike`反恐精英起源开发的带有动态菜单功能的`GDI`外部透视自瞄辅助，如下教程将为大家具体分析如何寻找基地址，以及透视算法如何实现，并应用已知数据实现一款透视自瞄辅助，该辅助已经在`Windows 10`系统下做过充分测试，稳定不闪烁。
